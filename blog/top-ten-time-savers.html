My Top 10 Time Savers

James McGaha • October 2, 2025

These are the top 10 tips & tricks that save me the most time and that I'd have the hardest time doing without.

  1. The SN Utils chrome extension
  2. Windows key + V to be able to copy/paste more than 1 thing at a time
  3. Using &sysparm_force_row_count=1&sysparm_group_sort=COUNTDESC while grouping (see Advanced List View Grouping)
  4. Adding dot-walked fields in list layouts
  5. Directly modifying the list view query
  6. Comprehensive search tool
  7. SN Utils's sn-scriptsync + GitHub Copilot in VS Code
  8. Getting around client side read-only, hidden, and mandatory field restrictions
  9. Service Portal widget debugging tricks
  10. Cancel transaction URL (/cancel_my_transaction.do)
